# Resume - Next.js

## Description

Resume built with Next.js 15+ (React).
Tested with Testing Library.
I tried using TailwindCSS to discover it.

## Development

By default app will be available at <http://localhost:3000/>

### With Docker

```sh
docker compose up -d
```

### With node and npm

Run in local using the following commands

```sh
npm ci
npm run dev
```
